# Processor Suffixes Reference

## Intel Processors

### Laptops

| Suffix  | Description                        |
|---------|------------------------------------|
| U       | Ultra-Low Power                    |
| Y       | Extremely Low Power                |
| H       | High Performance Graphics          |
| HQ/HK   | High-Performance, Unlocked (Mobile)|

### Desktops

| Suffix  | Description                        |
|---------|------------------------------------|
| C/K     | Unlocked Multiplier                |
| F       | No Integrated Graphics             |
| G       | Iris Xe Graphics                   |
| T       | Power-Optimized                    |
| X       | Extreme Performance                |
| E       | Embedded                           |
